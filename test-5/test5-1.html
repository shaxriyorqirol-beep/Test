Text decoration.

Text decoration bu matnning ko'rinishini o'zgartiruvchi xususiyatdir. Va u quyidagi qiymatlarni olishi mumkin:

  • underline - matn ostiga chiziq chizadi;
  • overline - matn ustiga chiziq chizadi;
  • line-through - matn o'rtasidan chiziq chizadi;
  • none - hech qanday dekoratsiya qo'llanmaydi.

Pseudo-elements.

Pseudo-elements bu CSSda maxsus elementlarni yaratish uchun ishlatiladi. Eng ko'p ishlatiladigan pseudo-elementlar quyidagilar:

  • :hover - foydalanuvchi element ustida sichqonchani tushirganda ishlaydi;

Pseudo-elements 2.

Pseudo-elements bu CSSda maxsus elementlarni yaratish uchun ishlatiladi. Eng ko'p ishlatiladigan pseudo-elementlar quyidagilar:

  • :active - foydalanuvchi elementni bosganda ishlaydi;

Qoidalarni guruhlash.

CSSda qoidalarni guruhlash orqali bir nechta elementlarga bir xil uslublarni qo'llash mumkin. Buning uchun qoidalarni vergul bilan ajratib yozish kifoya. Masalan:

  • h1, h2, h3 { color: blue; }
  • p, li, span { font-size: 16px; }
